Food rescue made simpler

Match extra meals with people who need them before the food expires.

MealBridge is a beginner-friendly Java project that filters donations by food type, location, quantity, and urgency. The matching logic uses basic DSA concepts so the project stays practical without becoming overcomplicated.

DSA Concepts Used

ArrayList storage, linear search, filtering conditions, and bubble sort for urgency-based matching.

SQL Concepts Used

Tables for donations and receivers, insert records, select records, and filter using WHERE with ORDER BY.

Java Concepts Used

Classes, objects, methods, encapsulation, Scanner input, loops, conditionals, and menu-driven programming.
